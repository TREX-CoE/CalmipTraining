 ## Using MAQAO on Olympe
 1. Login into `olympe`
 2. Add the TREX modulefiles
 ```
 module use /usr/local/trex/modulefiles
 ```
 3. Load the MAQAO module
 ```
 module load maqao/2.16.0
 ```
